# Project 3: AI Recommendation Logic

## Overview

This project is a simple AI Recommendation System developed as part of the DecodeLabs Artificial Intelligence Internship Program.

The system takes user interests as input and provides relevant recommendations based on predefined categories. It demonstrates the core concepts of recommendation systems, pattern matching, and decision-making logic.

---

## Objective

To create a recommendation system that:

* Accepts user preferences or interests
* Matches user input with available categories
* Provides relevant recommendations
* Handles invalid inputs gracefully
* Supports continuous interaction until the user exits

---

## Features

* User-friendly command-line interface
* Interest-based recommendation logic
* Multiple recommendation categories
* Continuous execution using loops
* Input sanitization using lower() and strip()
* Exit commands (exit, quit, bye)
* Fallback response for unknown interests

---

## How It Works

1. The user enters an area of interest.
2. The system checks whether the interest exists in the recommendation database.
3. If a match is found, relevant recommendations are displayed.
4. If no match is found, the user receives an appropriate message.
5. The program continues running until the user enters an exit command.
